PHP is one of the most widely used server side scripting language for web development. PHP is so popular because it's very simple to learn, code and deploy on server.

PHP is cross platform, hence you can easily develop and move/deploy your PHP code/project to almost all the major operating systems like Windows, Linux, Mac OSX etc. All the popular web hosting services support PHP. Also the web hosting plans for PHP are generally the amongst the cheapest plans becasue of its popularity.

It can be used to create Web applications like Social Networks(Facebook, Digg), Blogs(Wordpress, Joomla), eCommerce websites(OpenCart, Magento etc.) etc. Wordpress is one of the most used blogging(CMS) platform in the World and Laravel is most popular framework in PHP.
